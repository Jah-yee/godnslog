## 七、间距与尺寸规范

### 7.1 间距系统 (8px 基准)

```
4px  = 0.5x (极小间距)
8px  = 1x   (小间距)
16px = 2x   (中间距 - 默认)
24px = 3x   (大间距)
32px = 4x   (超大间距)
48px = 6x   (区块间距)
```

### 7.2 组件尺寸

```
按钮:
- Small:  32px 高
- Medium: 40px 高 (默认)
- Large:  48px 高

输入框:
- Small:  32px 高
- Medium: 40px 高 (默认)
- Large:  48px 高

卡片:
- Small:  padding 16px
- Medium: padding 24px (默认)
- Large:  padding 32px

表格行:
- Compact: 40px 高
- Normal:  56px 高 (默认)
- Loose:   64px 高
```

## 八、响应式断点

```
xs: < 640px   (手机竖屏)
sm: 640-768px (手机横屏/小平板)
md: 768-1024px (平板)
lg: 1024-1280px (桌面)
xl: > 1280px  (大桌面)

响应式策略:
- xs/sm: 侧边栏变为抽屉，表格变为卡片列表
- md:    侧边栏可收起，表格保持
- lg/xl: 侧边栏展开，完整布局
```
