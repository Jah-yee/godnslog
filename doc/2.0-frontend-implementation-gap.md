## API集成检查

### Dashboard所需API

| API端点 | 状态 | 用途 |
|---------|------|------|
| `GET /api/v2/cases?status=active&limit=1` | ✅ | 活跃Case数 |
| `GET /api/v2/interactions/stats` | ❌ 需开发 | 今日命中统计 |
| `GET /api/v2/interactions?today=true` | ⚠️ 需扩展 | 今日交互 |
| `GET /api/v2/payloads?status=deployed&limit=1` | ✅ | 活跃Payload数 |
| `GET /api/v2/interactions/trend?range=24h` | ❌ 需开发 | 趋势数据 |

### Case详情页所需API

| API端点 | 状态 | 用途 |
|---------|------|------|
| `GET /api/v2/cases/:id` | ✅ | Case详情 |
| `GET /api/v2/cases/:id/payloads` | ❌ 需开发 | Case关联Payloads |
| `GET /api/v2/cases/:id/interactions` | ❌ 需开发 | Case关联Interactions |
| `GET /api/v2/payloads/:id/interactions` | ❌ 需开发 | Payload命中记录 |

## 性能优化清单

- [ ] **虚拟列表**: Interaction列表超过100条时使用虚拟滚动
- [ ] **数据预取**: 路由切换时预取详情页数据
- [ ] **防抖搜索**: 搜索输入500ms防抖
- [ ] **分页加载**: 表格默认20条/页，最大100条/页
- [ ] **图片优化**: 使用Next.js Image组件
- [ ] **代码分割**: 大页面使用dynamic import
